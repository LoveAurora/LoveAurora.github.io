¶🙌 顺序栈 🙌

  • Push()函数判满 -> 顺序栈用的是数组，下标从 0 开始，top = MaxSize-1 ; 栈满 ; 🚀
  • Pop()函数判空 -> 顺序栈用的是数组，下标从 0 开始，top = -1;栈空; 🚀
  • Print()函数遍历打印完全 -> top 相当于数组的下标，i<S.top 才能把栈遍历完全。🚀
